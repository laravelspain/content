# Laravel Spain. Datos

Repositorio de datos públicos en JSON que alimentan https://laravelspain.com.

## Estructura

- `events.json` . Eventos y conferencias (Laravel, PHP, dev web)
- `packages.json` . Paquetes destacados del ecosistema Laravel
- `jobs.json` . Ofertas de empleo Laravel en España (actualmente vacío)
- `blog/` . Artículos del blog en Markdown con frontmatter YAML
- `README.md` . Documentación del repositorio

## Convenciones

- Idioma: español para todo el contenido (descripciones, README, commits)
- No usar "—" (em dash). Usar "." como separador
- Los JSON usan arrays de objetos, sin wrapper
- Campos de color CSS siguen el patrón Tailwind: `bg-{color}/10 text-{color}`
- El usuario prefiere tono informal pero profesional, sin emojis
